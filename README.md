# Juego-Beta

## Requisitos

- **Godot Engine 4.x** (o la versión compatible con el proyecto)
  - Descarga Godot desde: [https://godotengine.org/download](https://godotengine.org/download)

## Instalación

### 1. Descargar el repositorio de GitHub

Clona el repositorio del proyecto con el siguiente comando:

```bash
git clone https://github.com/SantiagoCumbal/Juego_beta.git
```

O descarga el proyecto como archivo ZIP desde GitHub y descomprímelo en tu computadora.

### 2. Abrir el proyecto en Godot

1. Abre **Godot Engine**
2. Haz clic en **Importar**
3. Navega hasta la carpeta donde descargaste el repositorio
4. Selecciona el archivo `project.godot`
5. Haz clic en **Importar y Editar**

### 3. Ejecutar el juego

Una vez que el proyecto esté abierto en Godot:

- Presiona **F5** o haz clic en el botón **▶ Play** en la esquina superior derecha para ejecutar el juego
- Para probar escenas individuales, abre la escena deseada y presiona **F6**

## Características del Juego

- Múltiples personajes jugables (Arquera, Caballero, Robot)
- Sistema de niveles progresivos
- Boss final
- Sistema de música y efectos de sonido
- Coleccionables y consumibles

## Notas

- **Compatibilidad**: Asegúrate de usar una versión de Godot compatible con el proyecto
- Si encuentras problemas al importar, verifica que todos los archivos `.import` estén presentes
